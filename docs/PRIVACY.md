# Privacy

## 12. Privacy First — Prinsip Eksplisit

Bukan preferensi, ini konstrain: **V Campfire tidak mengirim data apa pun keluar device secara default.** Konkretnya:

- Tidak ada telemetry, analytics, atau usage tracking terpasang secara default — titik.
- Tidak ada crash reporting otomatis ke layanan luar (konsisten dengan Bagian 7).
- Kalau suatu hari ada fitur online (cloud sync, AI assistant yang manggil API luar), fitur itu **wajib opt-in eksplisit**, bukan default aktif — dan harus jelas di UI data apa yang keluar device saat fitur itu dinyalakan.
- Kalau project ini open source, `docs/PRIVACY.md` yang menyatakan ini secara eksplisit adalah dokumen kepercayaan pertama yang dicek orang — worth ditulis sejak awal, bukan ditambahkan reaktif kalau ada yang nanya.

# Roadmap

## Feature scope

## 2. Feature Grouping — MVP vs Future vs Cut

### MVP (v0.1 – v0.3)
- App shell: sidebar nav, routing, design system dasar
- `tracked_items` (project + game unified): checklist, progress %, status, notes, gallery
- Focus timer: pomodoro + break reminder + basic stats
- Journal harian (done today / blockers / target besok)
- Sticky notes sederhana
- Musik: local file playback + 1 ambient sound layer, persistent across halaman
- Dashboard statistik dasar (jam fokus, checklist selesai, streak project)

### Future (v0.4 – v1.0+)
- Habit tracker dengan streak
- Workspace preset (bukan full widget engine)
- Playlist management + multi-layer ambient mixing
- Timeline visual untuk tracked items
- Charts / year-in-review untuk stats
- Backup/export lokal (zip berisi db + assets)

### Cut sepenuhnya (untuk sekarang)
- Version/diff system untuk project
- Full customizable widget layout engine
- Cloud sync (offline-first tetap default; kalau nanti mau sync, itu keputusan terpisah dengan implikasi auth/server yang gak kecil)
